## Installation

Install CianAPi with git
```bash
git clone https://github.com/NoneNameDeveloper/CianAPi
```

Go to directory
```bash
cd CianApi
```

Install requirements
```bash
pip3 install -r requirements.txt
```

Now you can develop your project  
#

## Usage/Examples
```python

# getting items list

    price_to=3_000_000
)
```
